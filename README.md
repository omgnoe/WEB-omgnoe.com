## 📖 About

Personal developer portfolio showcasing mobile apps, web projects, and cybersecurity services. Built with a cyberpunk-inspired glassmorphism design featuring animated backgrounds, terminal aesthetics, and crypto wallet integration.

Based in **Luxembourg** 🇱🇺

---

## ✨ Features

- 🎨 **Cyberpunk Glassmorphism** – Animated grid overlay, scan lines, floating glow orbs
- 💻 **Terminal Aesthetics** – Orbitron & JetBrains Mono fonts, code-style section tags
- 📱 **Mobile-First** – Fully responsive design with hamburger navigation
- 💰 **Wallet Integration** – Phantom (Solana) & MetaMask (Ethereum) donation support
- 🔒 **Privacy-Focused** – Obfuscated email protection
- ⚡ **Lightweight** – Pure HTML/CSS/JS, no frameworks
