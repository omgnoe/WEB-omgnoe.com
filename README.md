# omgnoe.com

Personal portfolio website showcasing mobile apps, web projects, and cybersecurity services based in Luxembourg.

## Tech Stack

- HTML5 / CSS3 / JavaScript
- Glassmorphism UI with cyberpunk aesthetics
- Mobile-first responsive design
- Phantom & MetaMask wallet integration

## Projects Featured

**Apps:** EchoChat, Budget Buddy, SellOutAI, Eurorecup  
**Websites:** Barbertruck.lu, O2Nails.lu, Nexa.lu, Absaar.com

## Setup

1. Clone the repo
2. Add your logo to `assets/logo.png`
3. Deploy to any static hosting
