EchoChat

// ZERO-KNOWLEDGE ENCRYPTED MESSENGER

Privacy isn't a feature – it's the foundation. No message storage, no metadata, no server logs. Your conversations exist only on your device.

Official Page GitHub

The Problem

Most "secure" messengers still store metadata: who communicated when with whom, IP addresses, device IDs. This data can be seized, hacked, or requested by authorities. Even if messages are encrypted – metadata often reveals more than the content itself.

The Solution

EchoChat was built from the ground up with zero-knowledge principles. The server sees only encrypted blobs – never plaintext. No phone number or email required. Sessions auto-expire after 3 days.

Server doesn't know: Your messages, name, contacts, IP, online status
Server only knows: Encrypted packets, temporary tokens (deleted after 3 days)

Features

E2E Encryption

X25519 + AES-256-GCM military-grade encryption

Zero-Knowledge

Server sees only encrypted blobs

Anonymous

No phone, no email required

Auto-Expire

Sessions expire after 3 days

QR Sharing

Add friends instantly via QR code

EU Server

GDPR protected infrastructure

Tech Stack

Flutter Dart Node.js WebSocket X25519 AES-256-GCM
